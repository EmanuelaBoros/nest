Otsu Thresholding Method

This plugin implements the Otsu thresholding method for image processing and it was ported from the open source library ImageJ.

The histogram is divided in two classes and the inter-class variance is minimized. This plugin outputs a thresholded image with the selected threshold applied on the entire image. The between class variance is computed and the threshold with the maximum between class variance also has the minimum within class variance,used for finding the best threshold in a simpler and better approach.

Parameters Used
For most filters, the following parameter should be selected:
  1. Source Band: All bands (real or virtual) of the source product. User can select one or more bands for producing thresholded images. If no bands are selected, then by default all bands will be selected. For complex product, only the intensity band can be selected.
