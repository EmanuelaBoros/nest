Parameters Used


The following parameters should be selected (see Figure 1 for example):

  1. Source Band: All bands (real or virtual) of the source product. User can select one or more bands for producing filtered images. If no bands are selected, then by default all bands will be selected. For complex product, only the intensity band can be selected.
  2. Iterations: the number of iterations
  3. Step: The step to display intermediate results
  4. Gradient Threshold: Threshold to find edges
  5. Maximum Displacement: Maximum displacements of points at each iteration
  6. Maximum Distance: Maximum distance to find and edges (used for growing or shrinking snakes)
  7. Regularization Factor: regularization value

First of all, select the region where the active contour will be initialized.


Figure 2. Selected Rectangle


Next, the image is shown with a ROIManager from ImageJ. Select a ROI by creating a rectangle on the image and the press Add from the ROIManager.


Figure 3. Selected Rectangle


The active contour is then initialized and all the steps are shown.
