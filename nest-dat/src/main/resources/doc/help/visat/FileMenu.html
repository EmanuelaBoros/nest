File Menu

DAT's File Menu

New Project

Creates a new Project for structuring data products, graphs and stacks.

Load Project

Loads an existing Project stored on the hard disk.

Import Browser  

Opens the Import Browser for data product previewing, opening or importing.

Save Project As...

Saves the current Project to the hard disk as a new file.

Close Project

Closes the currently opened Project.

Open (Control-O)  

Opens one or multiple data products stored on the hard disk in DAT. You can select more than one product, e.g. by pressing CTRL and selecting the desired products with the mouse.

Close (Control-W)

Closes the currently selected product. This commands is also available in the context menu when you right-click over the open product list.

Close All

Closes all products currently open in DAT. This commands is also available in the context menu when you right-click over the open product list.

Save (Control S)  

Saves the currently selected product in the BEAM-DIMAP format. If the product still does not have a file location, then you are asked to specify a storage location for the product. Note that save process can be time consuming for huge products.

Save As  

Saves the currently selected product to a different location. You are asked to specify a new storage location for the selected product.

Sessions

Open Session (Control-Shift-O)

Opens a session file stored on the hard disk. A session is termed the collection of all products and image views opened in the DAT, including any layers added to a view. When a session file is opened, a previously saved session is restored. Note that all open products will be closed before a session file is opened.

Close Session (Control-Shift-W)

Closes all products and image views currently open in the DAT. Note that changes made during a session are not written to an associated session file. In order to write the changes to a session file the session must be saved explicitly.

Save Session (Control-Shift-S)

Saves the current session to the associated session file. If the current session is not associated with a session file, you are asked to specify a storage location for the session file.

Save Session As

Saves the current session to a new session file. You are asked to specify a storage location for the new session file.

Import/Export

Product Readers

The list of available plug-in readers for supported products.

Product Writers

The list of available plug-in writers for supported products.

Exit

Automatically stores all preferences, last visited directories frame size and location and then exits the DAT application.
