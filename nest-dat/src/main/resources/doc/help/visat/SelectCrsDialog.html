Select CRS Dialog

One of the listed Coordinated Reference Systems (CRS) can be selected. The list is defined by the European Petroleum Survey Group (EPSG).

Filter: In the text box you can type in a word to filter the list of available Coordinate Reference Systems, the list beneath will be updated immediately.
Well-Known Text (WKT): This text area displays the definition of the CRS as Well-Known Text.
