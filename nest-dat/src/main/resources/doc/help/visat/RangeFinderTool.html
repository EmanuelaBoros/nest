Range Finder Tool

In the tools tool bar you can find the Range Finder Tool   . This tool allows you to measure the distance in km between two points of a displayed band.

On the first click in the image view you set the starting point. With every next click you set additional points. To get the measurement results click double to the end point. The distance will be displayed in a dialog box. Click on "Details..." to get more information about the result.
