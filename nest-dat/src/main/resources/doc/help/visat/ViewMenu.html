View Menu

DAT's View Menu

The DAT view menu provides general switches to visualisation tools, dialogs and overlays.

Show No-Data Overlay  

Toggles the overlay of a no-data mask over the band image view. The overlay properties can be modified at the no-data overlay page in the preferences dialog

Show Graticule Overlay  

Toggles the overlay of a graticule (geographic lat/lon grid) over the current image. The graticule overlay properties can be modified in the preferences dialog.

Show Pins 

Toggles whether or not to display the pins defined for the current product in the current image view.

Tool Windows

This menu item provides a list of available tool windows which you can activate.

Tool Bars

This menu item provides a list of available tool bars which you can activate.

Manage Layout

This menu consists of items to manage the current layout of tool windows.

Status Bar

Here you toggle the visibility of the status bar, which is located in the lowest area of the DAT main window. The status bar is used to display tool tips, the RAM consumption of DAT and a button where you are able to force a garbage collection (i.e. free unused memory). Deselecting the check box here will remove the status bar from the main DAT window (and vice versa).

Open Image View

Opens an image view for the selected band or tie point grid in the product browser. This menu entry is disabled when no band is selected.

Open RGB Image View

Opens a RGB image view for the selected product in the product browser.
