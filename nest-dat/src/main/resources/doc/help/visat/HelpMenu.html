Help Menu

DAT's Help Menu

The DAT Window Menu contains:

  •   Help Topics
    Opens the Help.
  • Test Connectivity
            Test if FTP access for the SRTM and Orbit file retrieval is ok
  • Report A Bug
            Opens a new email to send to the development team reporting a bug. This also includes some information about the version of the software, Java and operating system.
  • Request A Feature
            Opens a new email to send to the development team requesting a new feature that could be implemented.
  •   About...
    Opens the 'About DAT' window.
