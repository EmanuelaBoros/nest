The Settings Dialog


Settings

The Settings Dialog allows you to customize the default data path directories. From this window it is possible choose the root directory in which the Toolbox looks for the default Digital Elevation models and the supported orbits files.

String values specified here can use variables pointing to other entries or environment variables. For example, the root demPath can hold the path to all your DEM files and the aceDEMDataPath could be ${demPATH}\ACE to specify the location of the ACE DEM files.

Changes made via the Settings dialog will be saved in a settings.xml file.
