Import GeoTIFF

This command allows to import the data of a GeoTIFF file.

GeoTIFF is an extension to the TIFF 6.0 specification. These image files contain additional information about their georeference and their spatial resolution. A wide range of projected coordinate systems are supported including UTM, US State Plane, and National Grids. For further information have a look at the following documentations:

  • GeoTIFF Homepage
  • TIFF Specification

Constraints of reading GeoTIFF product files

If a GeoTiff file was exported by NEST, the reader can recover information from a special tag used by the GeoTIFF writer.
This includes:

  • Name and type of the product.
  • Name, description and unit of each band.
  • The properties of no-data and scaling for each band.
  • The index coding of a band.
  • Virtual and filtered bands.

For features and limitations of the GeoTIFF import and constraints see the following list:

  • Only the first image of a GeoTIFF file is considered.
  • The size of images is limited to a pixel count of 232.
  • The GeoTIFF reader supports the projections available in NEST, model transformations and tie-points.
  • Color maps are considered and converted to an index coding.
