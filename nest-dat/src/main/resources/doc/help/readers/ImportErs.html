Import ERS1/2 SAR or ATSR

Import ERS Product in ENVISAT format

This option allows to import ERS1/2 products stored in the ENVISAT format into DAT.

The supported ERS1/2 product types are:

  • SAR_IMG_1P, SAR_IMM_1P, SAR_IMP_1P, SAR_IMS_1P, SAR_IM__BP products
  • ATSR and ATSR-2 TOA products

ERS products in ENVISAT format have extension .E1 for ERS 1 and .E2 for ERS 2
