Orthorectification Tutorial

Terrain Correction

Geocoding is the process of resampling the data to conform to a standard map projection with known coordinates. There are various levels of geocoding

  • Geocoded Ellipsoid Corrected (GEC)
The GEC product is a multi-look (speckle-reduced), ground range, system corrected image. It is precisely located and rectified onto a map projection, but not corrected for terrain distortion
  • Geocoded Terrain Corrected (GTC)
The GTC product is precisely located, corrected for terrain variations and rectified onto a map projection.
  • Radiometric Terrain Corrected (RTC)
The RTC product is precisely located, corrected for terrain variations, rectified onto a map projection and retro calibrated.

The Terrain Correction Operator will produce GTC and optionally RTC products. The Ellipsoid Correction RD and Ellipsoid Correction GG Operators will produce GEC products.

The Terrain Correction Operator should be used whenever DEM is available. The Ellipsoid Correction (RD and GG) should be used only when DEM is not available.


1. From the Geometry menu select Terrain Correction. This will call up the dialog for the Terrain Correction Operator.

2. Select your source bands.

3. Select the Digital Elevation Model (DEM) to use. You can select 30 second GETASSE30 or ACE DEMs if they are installed on your computer.

    Preferably, select the SRTM 3 second DEM which has much better resolution and can be downloaded as need automaticaly by NEST if you have an internet connection.

    Alternatively, you could also browse for an External DEM tile in any file format readable by NEST (GeoTIFF, HDF, NetCDF, GTOPO30, ...)

4. Select the interpolation methods to use for the DEM resampling and the target image resampling.

5. Optionally select to save the DEM as a band and the local incidence angle.

6. Optionlly select to apply Radiometric Correction to do a retro calibration of the resulting product. Note: Radiometric Correction is currently only supported for ASAR and ERS products.




7. Press Run to process the data.
