Frequently Asked Questions

General F.A.Q.

I can't open an ENVISAT product.

First, try to open the products with Enviview http://earth.esa.int/enviview/. If that works then it could be a version not supported by NEST. Let us know and we'll try to get it supported asap. If Enviview and NEST are not able to read the product, then it could be that the file is corrupt or compressed. 

Some users have had difficulty when decompressing files using Winzip. Deactivate the CR/LF translation option in winzip (this option is activated by default).
