NEST Overview

NEST Version 3C

Overview

The Next ESA SAR Toolbox (NEST) is used for reading, post-processing, analyzing and visualizing the large archive of data from ESA SAR missions including ERS1 & 2, ENIVSAT and eventually Sentinel-1, as well as third party SAR-data from JERS SAR, ALOS PALSAR, TerraSAR-X, Radarsat-1 and Radarsat-2. NEST will help the remote sensing community by handling ESA SAR products and complimenting existing commercial packages.

The main components of the NEST are:

  • DAT - A desktop application used for EO data visualisation, analysing and processing.
  • A rich collection of readers and writers to allow easy and effficient access to SAR data, abstracted interanl representation and output to common formats for compatibility with other software.
  • A set of scientific data processors running either from the command line or invoked by DAT.
  • A powerful and flexible graph processing framework to allow you to create you own processing chains.
  • A Java™ API which provides ready-to-use components for remote sensing related application development and plug-in points for new NEST extension modules.

Supported Product Formats


 NEST includes readers for the following data products:

  •     ENVISAT ASAR, MERIS, AATSR
  •     ERS AMI 1 & 2 (CEOS & Envisat format)
  •     JERS SAR
  •     Radarsat 1 
  •     DIMAP
  •     GETASSE30 DEM
  •     ACE DEM

Information about access to these products is given on the Data Sources page.


An API is provided with NEST to allow easy extension by users to add data ingestion of other formats and to support data formats of future missions.

Source Code

The complete NEST software has been developed under the GNU public license and comes with full source code in JavaTM.

Supported Platforms

NEST is programmed in pure Java™ to allow a maximum portability. The NEST software has been successfully tested under MS Windows™ XP® as well as under Linux and Solaris®; operating systems. NEST will be available for Mac OS X as soon as Apple supports the J2SE 6 platform.
