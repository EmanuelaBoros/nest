Data Access Details and Distribution Policy

  Instrument How to get data
1 MERIS

Data Access
MERIS data can be obtained by several means:

  • - download of L1b and L2 RR data of the last 7 days from the rolling archive/EWFS
  • - download of L1b and L2 RR data from 2002 until current from the Merci catalogue (one year of data missing)
  • - download of L1b FRS data over Europe of the last 7 days from the rolling archive/EWFS
  • - ordering L1b and L2 FR data from 2002 until current on CD/DVD from the EOLI-SA catalogue
  • - download MERIS binned Level 3 data from the MERIS Level3 demonstration products page
  • - reception of MERIS RR data in NRT using a telecommunication link (satellite dish) through the ESA DDS system (contact eohelp@esa.int to get access to the DDS system. See http://dwlinkdvb.esrin.esa.it/DDS/ for technical details).

MERIS RR data are systematically processed after acquisition up to Level 2 and put on the Rolling Archive. They remain there for download for 7 days. After this period they are transferred into the MERCI system. MERCI includes all consolidated MERIS L1b and L2 RR data from 2002 until current for download.

MERIS FR full swath data are systematically processed until L1b for European coverage and can be downloaded from the rolling archive (last 7 days). Historic FR data are processed on demand and delivered on media. They have to be selected in the EOLI-SA catalogue and ordered.

Data Policy
Download of MERIS RR L1b and L2 data and MERIS FRS data over Europe from the rolling archive/EWFS requires a registration with ESA http://eopi.esa.int/esa/esa?cmd=submission&aoname=lbr. More information about the registration can be found on http://eopi.esa.int/esa/esa?cmd=aodetail&aoname=lbr

Once a user has a registered project with EOPI for MERIS RR data access, download of MERIS RR L1b and L2 data from the MERCI system requires an informal application by Email to EOHELP: eohelp@esa.int. EOHELP sends back a user name and password. Ordering of data from the EOLI-SA catalogue and download of MERIS FR data (other than European coverage) from the rolling archive requires an accepted Cat-1 proposal. Proposal can be submitted from http://eopi.esa.int/esa/esa?cmd=submission&aoname=cat1. More information about Cat-1 proposal can be found on http://eopi.esa.int/esa/esa?e=XDXuWvAzMdf1mrTn3WnRZIs3yw9RTGP0TGGYmX3nY2plBLAAh

MERIS Binned Level 3 data can be downloaded without any registration.

MERIS Binned Level-3

MERIS Binned Level 3 data can be downloaded without any registration.

2 AATSR

Data Access
AATSR data of the last 7 days can be downloaded from the ESA Rolling Archive/EWFS.
Historic data can be ordered on media through the ESA EOLI-SA interface.
Reception of AATSR data in NRT using a telecommunication link (satellite dish) through the ESA DDS system (contact eohelp@esa.int to get access to the DDS system. See http://dwlinkdvb.esrin.esa.it/DDS/ for technical details).
AATSR data can be downloaded from the NERC data server.

Data Policy
Download of AATSR data from the rolling archive/EWFS requires a registration with ESA http://eopi.esa.int/esa/esa?cmd=submission&aoname=lbr. More information about the registration can be found on http://eopi.esa.int/esa/esa?cmd=aodetail&aoname=lbr

Ordering of data from the EOLI-SA catalogue requires an accepted Cat-1 proposal. Proposal can be submitted from http://eopi.esa.int/esa/esa?cmd=submission&aoname=cat1. More information about Cat-1 proposal can be found on http://eopi.esa.int/esa/esa?e=XDXuWvAzMdf1mrTn3WnRZIs3yw9RTGP0TGGYmX3nY2plBLAAh

NERC requests registration before the data can be downloaded: http://www.neodc.rl.ac.uk/?option=displaypage&Itemid=91&op=page&SubMenu=-1

10 MODIS

Data Access
MODIS data can be downloaded using an interactive Web interface or from an ftp site at NASA.

Data Policy

11 AVHRR/3

Data Access
AVHRR/3 data from NOAA are available through direct reception of the satellite data or via various http and ftp server. One example is the NOAA class server.
AVHRR/3 data from Metop-A (Eumetsat) are currently (April 2007) in commissioning phase. The data will be available by direct reception of the HRPT data stream and possibly ftp servers.

Data Policy
Data can be freely received using an HRPT acquisition system. They can be downloaded, e.g. from the NOAA class server, which requires a simple registration (name and Email).

12 TM Thematic Mapper

Data Access
Data are distributed through Eurimage and other data providers. A good online search and order facility is provided by Eurimage (inet). To use the inet catalogue a registration with Eurimage is required.

There is also the European dataset available from ESA via EOLI-SA (cat-1 project)

Data Policy
Landsat data are commercially available (Eurimage).
