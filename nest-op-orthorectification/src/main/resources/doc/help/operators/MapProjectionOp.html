Map Projection

Map Projection Operator

The Map Projection Operator applies a selected Map Projection to the input product and creates a new transposed output product. 




The following parameters are used by the Operator:

  1. Source Bands: All bands (real or virtual) of the source product. You may select one or more bands. If no bands are selected, then by default all bands will be processed.

  2. Map Projection: You can select one of the supported map projections here. For a description of the available map-projections, please refer to the chapter Available Map Projections.
